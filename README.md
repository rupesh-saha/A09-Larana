Larana - Medical Appointment Booking Platform



Live Site URL: https://larana-med.vercel.app/


Live Server URL: https://larana-server.vercel.app/

Larana is a premium, full-stack healthcare platform designed to connect patients with top medical professionals seamlessly. Built with a focus on modern UI/UX principles and robust backend security, it simplifies the process of finding specialists and managing medical appointments.

Key Features
Secure User Authentication: Integrated with Better Auth to provide seamless and secure Google and Email login options, protected by strict JWT backend verification.

Real-Time Smart Search: A highly optimised search system allowing users to instantly filter healthcare professionals by name, speciality, or hospital.

Interactive Booking System: Features a controlled, modal-based appointment scheduling flow that protects data integrity and provides immediate UI feedback via toast notifications.

Private User Dashboard: A dedicated, protected route where authenticated users can view their appointment history, safely manage bookings (Update/Delete), and securely update their profile information.

Premium Glassmorphism UI: Engineered with Next.js App Router, Tailwind CSS, and HeroUI v3 to deliver a highly responsive, modern, and visually striking user experience across all devices.
